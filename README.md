## Software Anaylse 
Ik wist dat ik aan het begin van de opdracht Unity ging gebruiken. Van alle engines en talen waar ik mee gewerkt heb de laatste 3 jaar heb ik de meeste ervaring met Unity en C#. De andere optie die ik had was met HTML, CSS en javascript te werken. Met HTML en CSS heb ik een klein beetje ervaring, maar javascript heb ik altijd een hele lastige taal gevonden. Daarbij ben ik niet van plan om javascript in de toekomst te gebruiken. Dus ook al zou ik wat van javascript leren, is het handiger dat ik hetzelfde idee leer, maar dan in Unity. Ook heb ik gekeken naar welke taal ik wil gebruiken voor de externe dialogue
